---
name: tarie design philosophy
description: Core product philosophy - completion over perfection, constrained format over blank canvas
type: project
---

tarieの根底にある思想: 自由度が高すぎて何も作れないよりも、型にハマっていても完成させる方がマシ。写真を並べるだけで1冊の本になるという割り切り。

**Why:** 白紙のキャンバスは自由だが、多くの人は何も作らずに終わる。制約があることで完成に至れる。
**How to apply:** 機能追加の判断基準にする。自由度を上げる機能よりも、完成までの摩擦を減らす機能を優先する。複雑なカスタマイズより、少ない操作で「それなりに良い」結果が出ることを重視する。
